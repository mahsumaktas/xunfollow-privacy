Privacy Policy

X Unfollow Tool — Last Updated: February 18, 2026

Overview

X Unfollow Tool is a browser extension that helps users manage their following list on X (formerly Twitter). This privacy policy explains how we handle your data.

Data Collection

We do NOT collect, transmit, or share any personal data.

All data generated by this extension is stored exclusively on your device using Chrome's local storage API (chrome.storage.local).

What Data is Stored Locally

The following data is stored only on your device:

  • User preferences — Theme, language, speed settings, selected mode
  • Whitelist — Usernames you choose to protect (entered by you)
  • Activity logs — Records of extension actions (up to 100 entries)
  • Unfollow history — Usernames that were unfollowed (up to 500 entries)
  • Statistics — Session counts, totals (unfollowed, scanned, skipped)
  • Onboarding state — Whether you have completed the first-run guide

Network Requests

This extension only communicates with:

  • x.com / twitter.com — To perform follow/unfollow actions and retrieve profile information on the pages you visit. These requests use your existing browser session cookies.

We do NOT send data to:

  • Any analytics or tracking services
  • Any external servers or APIs
  • Any third-party services
  • Any advertising platforms

Data Sharing

We do NOT share any data with third parties. There are no analytics, no telemetry, no tracking pixels, and no advertising SDKs in this extension.

Data Retention

  • All data persists locally until you clear it manually (via the extension UI or browser settings)
  • Uninstalling the extension removes all stored data
  • You can export/import your data at any time via the Settings tab

Your Rights

You have full control over your data:

  • View — All stored data is accessible via the extension UI
  • Export — Use the Export feature to download your data as JSON or CSV
  • Delete — Clear logs, history, or all data at any time
  • Uninstall — Removing the extension deletes all associated data
