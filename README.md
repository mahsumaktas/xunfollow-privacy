# X Unfollow Tool — Smart Following Manager

Chrome extension to bulk unfollow non-followers or everyone on X (Twitter).  
Smart speed control, whitelist protection & activity logs.

## Features

- 🔍 Detect non-followers and unfollow in bulk
- 🛡️ Whitelist protection — keep the accounts you care about
- ⚡ Smart speed control to avoid rate limits
- 📋 Activity logs for every action
- 🌍 Multi-language support (EN, ES, JA, PT-BR, TR)

## Install

1. Download or clone this repository
2. Open `chrome://extensions/` in Chrome
3. Enable **Developer mode**
4. Click **Load unpacked** and select this folder

## Privacy

This extension runs entirely in your browser. No data is collected, stored, or sent to any server.  
See [PRIVACY.md](PRIVACY.md) for details.
